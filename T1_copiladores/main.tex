\subsubsection*{c)}

<KW_IF, If>
<LEFTPAR, (>
<IDENT, a>

<IDENT, b>

<KW_WRITE, Write>
<LEFTPAR, (>
<STRING, "Analise lexica">
<RIGHTPAR, )>
<PONTO_VIRGULA, ;>

\section*{Aula 3 -- Autômatos finitos}

\subsection*{Exercício 1 -- Construção de autômatos}
Os slides da \texttt{Aula3} apresentam o seguinte bloco de exercícios:
\begin{enumerate}
  \item \( \{w \in \{a,b\}^* \mid w \text{ tem tamanho } 3\} \)
  \item \( \{w \in \{a,b\}^* \mid w \text{ tem tamanho maior que } 3\} \)
  \item \( \{w \in \{a,b\}^* \mid w \text{ tem como prefixo } aa\} \)
  \item \( \{w \in \{a,b\}^* \mid w \text{ tem como sufixo } aaa \text{ ou } bb\} \)
  \item \( \{w \in \{a,b\}^* \mid w \text{ tem tamanho múltiplo de } 3\} \)
  \item \( \{w \in \{0,1\}^* \mid \text{cada } 0 \text{ de } w \text{ é imediatamente seguido de, no mínimo, dois } 1\text{'s}\} \)
  \item \( \{w \in \{0,1\}^* \mid w \text{ não contém } 000 \text{ nem } 111\} \)
  \item \( \{w \in \{0,1\}^* \mid \text{os últimos três símbolos de } w \text{ não são } 000\} \)
  \item \( \{w \in \{a,b\}^* \mid w \text{ não contém } ab\} \)
  \item \( \{w \in \{a,b\}^* \mid w \text{ tem tamanho múltiplo de } 3 \text{ e não contém } ab\} \)
\end{enumerate}

Abaixo estão autômatos válidos para cada item. Em alguns casos, foi usado AFND para manter o desenho mais direto.
